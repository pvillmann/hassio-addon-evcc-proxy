# evcc Proxy Add-ons for Home Assistant

This repository contains Home Assistant add-ons.

## Add-ons

### evcc (Proxy)

Home Assistant Ingress reverse proxy add-on for externally hosted evcc instances.

### Install in Home Assistant

1. Open **Settings → Add-ons → Add-on Store**.
2. Click the menu (⋮) → **Repositories**.
3. Add: `https://github.com/pvillmann/hassio-addon-evcc-proxy`
4. Find **evcc (Proxy)** and install it.
